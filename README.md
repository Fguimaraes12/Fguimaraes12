## 💜 Sobre Mim

Sou desenvolvedor **frontend** cursando **Sistemas de Informação** na Universidade 7 de Setembro (Fortaleza, CE), com foco em construir produtos web bem projetados, performáticos e prontos para produção. Trabalho principalmente com **Next.js**, **TypeScript** e **React**, com atenção especial a gerenciamento de estado (`useReducer`, `useContext`), design systems consistentes e experiência do usuário.

Tenho me aprofundado em **desenvolvimento full stack**, expandindo para **Prisma**, **Supabase**, **PostgreSQL** e **FastAPI**, sempre buscando entender o sistema de ponta a ponta — do componente de UI à modelagem do banco de dados.

Gosto de aprender construindo: cada projeto novo nasce para dominar um conceito específico (hooks avançados, integrações de API, design systems), com iteração constante até chegar a um resultado de produção.

**🎯 Aberto para:**
- Vagas de **Frontend Developer Júnior**
- Projetos open source em React/Next.js
- Oportunidades de estágio em desenvolvimento web
